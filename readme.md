# This is the start of a bespoke, kanban/trello style migration tracker

Build Tracker is a WordPress plugin designed to streamline website build management. It allows teams to track, approve, and manage changes across website builds with an intuitive drag-and-drop workflow. Key features include:

- Create and manage multiple builds with versioned change tracking
- Track changes, approvals, and post-launch testing status
- Attach files, screenshots, and notes to each change item
- Visual progress bars and detailed dashboards for quick insights
- AJAX-driven, responsive, and user-friendly admin interface

Ideal for small agencies and teams that need structured, auditable workflows for website updates and deployments.
